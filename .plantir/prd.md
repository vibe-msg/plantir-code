React 기반으로 개발되는 사주 앱으로, 사용자가 자신의 이름, 한문명, 생년월일, 태어난 시간 등 사주에 필요한 정보를 입력할 수 있으며 해당 정보를 통해 오늘의 운세를 제공하는 기능을 제공합니다.
사용자는 한달 간 일일별 운세를 탐색하고, 개별 운세 페이지에서 스와이프를 통해 다른 운세로 이동할 수 있습니다.
데이터베이스는 사용하지 않으며 Node.js의 메모리에 저장하여 처리합니다.

상세 기술 스택은 다음과 같습니다.

- Next.js(App Router)
- TypeScript
- Jotai (https://jotai.org/)

프로젝트명은 영문으로 ounse이며, 프로젝트 폴더명으로 사용할 수 있습니다. 한글명은 오운세입니다.
